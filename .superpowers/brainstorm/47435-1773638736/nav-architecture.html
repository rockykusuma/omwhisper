Which direction?

✓ Unify into one Home tab

Remove the Transcribe nav item. Home becomes the single action screen: record button + stats + mic/model + recent recordings preview. Sidebar shrinks from 6 items to 5.

  • Matches how tray apps actually work (SuperWhisper, Whisper Transcription)
  • Record button lives where context lives — no tab-switching
  • Recent recordings fills the empty space meaningfully
  • "What's New" block can be removed (gets stale fast)

Keep separate tabs, enrich Transcribe

Home stays as-is. Transcribe gets filled with Option A/B/C from the previous screen.

  • More separation of concerns — Home = overview, Transcribe = workspace
  • Still requires navigating to Transcribe to see the live transcript
  • Duplicates model/mic info across both tabs
