# Installation Guide — Web App Page Content

Use this content for the `/beta-install` page on omwhisper.in.

---

## Page Content

### Heading
```
Installing OmWhisper Beta
```

### Subheading
```
Get up and running in 2 minutes.
```

---

### Step 1 — Download

Click the download button to get the `.dmg` file.

> **macOS 13 (Ventura) or later required. Apple Silicon (M1/M2/M3) only for now.**

---

### Step 2 — Install

1. Open the downloaded `.dmg` file
2. Drag **OmWhisper** into your **Applications** folder
3. Eject the disk image

---

### Step 3 — Launch

Open **Finder → Applications** and double-click **OmWhisper**.

---

### Step 4 — Grant Microphone Permission

macOS will ask for microphone access on first launch. Click **Allow**.

If you missed it or denied it:
**System Settings → Privacy & Security → Microphone → enable OmWhisper**

---

### Step 5 — Grant Accessibility Permission (for Auto-Paste)

Auto-paste types your transcription directly into whatever app you're using. It requires Accessibility access.

1. Go to **System Settings → Privacy & Security → Accessibility**
2. Click the **+** button
3. Add **OmWhisper** and make sure the toggle is **on**

> Without this, transcription still works — it's just copied to your clipboard instead of pasted automatically.

---

### You're ready!

Press **⌘⇧V** from any app to start dictating. Your words will appear and paste automatically when you stop.

---

### Sending Feedback

Use **Settings → About → Send Feedback** inside the app. Reports go directly to the team.

---

### Uninstalling

Drag OmWhisper from Applications to Trash. To remove all data:
```
~/Library/Application Support/com.omwhisper.app/
```
