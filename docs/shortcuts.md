# Keyboard Shortcuts

All hotkeys can be customised in **Settings → General**.

---
layout: default

## Global Hotkeys

These work system-wide even when OmWhisper is not the focused window.

| Action | macOS (default) | Windows (default) |
|--------|----------------|-------------------|
| Toggle recording | `Cmd+Shift+V` | `Ctrl+Shift+V` |
| Smart Dictation | `Cmd+Shift+B` | `Ctrl+Shift+B` |
| Polish last text | `Cmd+Shift+P` | `Ctrl+Shift+P` |

---

## Push-to-Talk Keys (macOS only)

In Push-to-Talk mode, you can use a single key instead of a chord:

- `Fn`
- `Caps Lock`
- `Right Option`
- `Right Control`

Configure in **Settings → General → Recording Mode → Push to Talk**.
